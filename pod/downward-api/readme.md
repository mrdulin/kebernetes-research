# 在容器中获取 pod 信息(Downward API)

### 环境变量方式：将 Pod 信息注入为环境变量

```bash
☁  kubernetes-labs [master] ⚡  k logs dapi-test-pod
KUBERNETES_SERVICE_PORT=443
KUBERNETES_PORT=tcp://10.96.0.1:443
TOMCAT_SERVICE_PORT_8080_TCP=tcp://10.98.25.80:8080
HOSTNAME=dapi-test-pod
SHLVL=1
HOME=/root
MY_POD_NAMESPACE=default
MY_POD_IP=10.1.0.26
KUBERNETES_PORT_443_TCP_ADDR=10.96.0.1
PATH=/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin
KUBERNETES_PORT_443_TCP_PORT=443
KUBERNETES_PORT_443_TCP_PROTO=tcp
TOMCAT_SERVICE_PORT_8080_TCP_ADDR=10.98.25.80
TOMCAT_SERVICE_SERVICE_HOST=10.98.25.80
KUBERNETES_PORT_443_TCP=tcp://10.96.0.1:443
KUBERNETES_SERVICE_PORT_HTTPS=443
TOMCAT_SERVICE_PORT_8080_TCP_PORT=8080
KUBERNETES_SERVICE_HOST=10.96.0.1
TOMCAT_SERVICE_PORT_8080_TCP_PROTO=tcp
PWD=/
TOMCAT_SERVICE_SERVICE_PORT=8080
TOMCAT_SERVICE_PORT=tcp://10.98.25.80:8080
MY_POD_NAME=dapi-test-pod
```
