# kubernetes-research

A nodejs application for kubernetes research

## 说明

执行`eval $(minikube docker-env)`命令，`docker ps`就可以查看在 minikube 的 VM 中运行的 container 了

`minikube start` - 在本机启动 kubernetes 集群
